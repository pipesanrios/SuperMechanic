== Description ==
Super Mechanic helps automotive workshops and dealerships manage daily operations from one place.

Core capabilities include:
- Client and vehicle management
- Operational process tracking
- Maintenance and delivery-related workflows
- Quotes, invoices, and payment records
- CRM opportunity and follow-up management
- Appointment and calendar operations

== Installation ==
1. Upload the `super-mechanic` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Open "Super Mechanic" in wp-admin and configure your initial settings.

== Frequently Asked Questions ==
= Is this plugin intended for workshops only? =
It is built for automotive workshops and dealership service operations.

= Does it support appointments and CRM tracking? =
Yes. The plugin includes operational scheduling plus CRM opportunity/task tracking.

= Does it require custom database tables? =
Yes. Required tables are created by the plugin during activation.
