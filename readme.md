# Super Mechanic

Super Mechanic is an operations and CRM platform for automotive workshops and dealerships, built on WordPress.

It centralizes service operations, vehicle lifecycle tracking, customer management, and commercial workflows into a single system.

---

## What Problem It Solves

Most workshops operate with disconnected tools:

- spreadsheets
- WhatsApp
- manual tracking
- lost follow-ups

Super Mechanic replaces that with a structured, process-driven system.

---

## What The System Does

- Tracks every vehicle through defined workflows
- Manages clients and full vehicle history
- Controls maintenance, delivery, and paperwork processes
- Handles quotes, invoices, and payments
- Organizes CRM pipeline and follow-ups
- Schedules tasks and appointments
- Generates operational visibility across the business

---

## How It Works

The system is built around **processes**.

Each vehicle moves through structured workflows such as:

- Maintenance
- Pre-delivery
- Paperwork

Each process contains:

- steps
- tasks
- deadlines
- alerts

This creates full operational control and traceability.

## Quick Installation

1. Copy the plugin folder into `wp-content/plugins/super-mechanic`
2. Activate **Super Mechanic** from WordPress Admin → Plugins
3. Open the plugin menu and complete setup
