PHASE: 42C
NAME: Acciones Masivas Seguras

==================================================
OBJETIVO
==================================================

Permitir ejecucion manual de acciones masivas controladas
sobre entidades operativas soportadas desde dashboard.

==================================================
SCOPE
==================================================

- detectar grupos masivos operativos
- ejecutar acciones masivas manuales con validacion estricta
- limitar 42C a entidad crm_task
- mantener trazabilidad y feedback en dashboard

==================================================
OUT OF SCOPE
==================================================

- cron
- autoejecucion
- reglas dinamicas
- nuevas tablas
- cambios en CRM Pipeline core
- acciones sobre procesos/citas

==================================================
ALLOWED FILES
==================================================

- includes/dashboard/class-workload-service.php
- includes/dashboard/class-admin-dashboard-controller.php
- includes/crm/class-crm-task-service.php
- includes/crm/class-crm-task-repository.php
- docs/contracts/42C.md
- docs/contracts/validation/42C-validation.md

==================================================
FORBIDDEN FILES
==================================================

- includes/modules/*
- schema/database
- includes/crm/class-crm-pipeline-*
- includes/processes/*
- includes/appointments/*
- invoices/payments core

==================================================
ACCEPTANCE CRITERIA
==================================================

- existe deteccion de grupos para acciones masivas
- existe ejecucion segura para bulk_resolve y bulk_reassign en crm_task
- validaciones de seguridad aplicadas
- sin nuevas tablas
- sin regresion en 40A-42B

==================================================
VALIDATIONS REQUIRED
==================================================

- php lint
- QA runner
- runtime manual

==================================================
DOCS TO UPDATE
==================================================

- docs/contracts/validation/42C-validation.md

==================================================
TECHNICAL DEBT
==================================================

- 42C solo soporta crm_task
- sin acciones masivas sobre procesos/citas
- sin configuracion de umbrales por UI
