# TASK CONTRACT — 56P13-C First Connector Prototype Mock

## Objective

Implement a first mock/local inventory connector prototype to validate the external connector architecture without real providers, external APIs, OAuth, scheduled sync, or admin UI.

## Scope

- mock/local inventory connector runtime under `includes/integrations/inventory-connectors/*`
- provider-agnostic adapter flow
- normalized payload generation according to `docs/INVENTORY_CONNECTOR_CONTRACT.md`
- dry-run result without database writes
- sync simulation result without provider/API calls and without real imports
- documentation updates for current state, QA, and task log

  - `normalize_item()`
