PHASE: 42D
NAME: Centro de Accion Operativa

==================================================
OBJETIVO
==================================================

Unificar en un solo bloque del dashboard la ejecucion operativa manual
de acciones asistidas (42A), reasignacion (42B) y acciones masivas (42C),
manteniendo una vista priorizada con sugerencias y flags criticos.

==================================================
SCOPE
==================================================

- crear bloque UI "Centro de Accion Operativa" en dashboard admin
- consolidar acciones desde metodos existentes sin recalculo
- reutilizar handlers, nonce y capability ya implementados
- mantener bloques previos y agregar centro como capa superior

==================================================
OUT OF SCOPE
==================================================

- nuevas reglas de negocio
- cambios en umbrales operativos
- cambios en CRM Pipeline
- nuevos endpoints
- nuevas tablas o cron

==================================================
ALLOWED FILES
==================================================

- includes/dashboard/class-admin-dashboard-controller.php
- includes/dashboard/class-workload-service.php (solo lectura / consumo)
- assets/css/admin.css
- docs/contracts/42D.md
- docs/contracts/validation/42D-validation.md

==================================================
FORBIDDEN FILES
==================================================

- includes/modules/*
- includes/crm/class-crm-pipeline-*
- schema/database
- includes/processes/*
- includes/appointments/*

==================================================
ACCEPTANCE CRITERIA
==================================================

- existe render_operational_action_center(...) en dashboard controller
- el centro consume metodos existentes de 42A/42B/42C/41E sin duplicar logica
- el centro permite ejecutar acciones desde un solo lugar reutilizando handlers existentes
- no hay cambios de reglas de negocio
- no hay regresion funcional en bloques 40A-42C

==================================================
VALIDATIONS REQUIRED
==================================================

- php lint
- QA runner
- runtime manual
