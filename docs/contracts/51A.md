PHASE: 51A
NAME: Licensing System Base

==================================================
OBJETIVO
==================================================

Implementar una base de licenciamiento para controlar
activación, estado y validez del producto por instalación.

==================================================
SCOPE
==================================================

- modelo de licencia
- installer + repository + service
- validación local de licencia
- vinculación a dominio/sitio
- estado de licencia visible en admin

==================================================
OUT OF SCOPE
==================================================

- servidor remoto de licencias
- validación online
- facturación
- UI avanzada de billing
- límites por plan
- bloqueo agresivo del sistema

==================================================
CRITERIO DE IMPLEMENTACIÓN
==================================================

- validación local simple
- dominio derivado del sitio actual
- no bloquear completamente el sistema todavía
- mostrar estado visible en admin
- dejar preparado para validación remota futura

==================================================
COMPORTAMIENTO INICIAL
==================================================

- si no hay licencia:
  - sistema sigue funcionando
  - mostrar estado "inactive"
  - admin notice claro
- si licencia no coincide con dominio:
  - marcar inválida para sitio actual
  - no bloquear todavía
- si expirada/revocada:
  - estado visible
  - sin enforcement agresivo aún
