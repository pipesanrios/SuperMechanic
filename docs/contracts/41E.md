PHASE: 41E
NAME: Consola de Automatización

==================================================
OBJETIVO
==================================================

Crear una consola unificada en dashboard que permita
visualizar el estado completo del sistema automático.

==================================================
SCOPE
==================================================

- consolidar información de:
  - flags (41A)
  - escalamiento (41B)
  - sugerencias (41C)
  - asignaciones (41D)

- mostrar estado global del sistema
- permitir lectura clara del sistema automático

==================================================
OUT OF SCOPE
==================================================

- ejecutar acciones
- modificar datos
- nuevas tablas
- cron
- tocar CRM Pipeline
- lógica nueva de negocio

ALLOWED FILES
==================================================

- includes/dashboard/class-workload-service.php
- includes/dashboard/class-admin-dashboard-controller.php
- assets/css/admin.css (solo si es necesario)
- docs/contracts/41E.md
- docs/contracts/validation/41E-validation.md

OBJETIVO FUNCIONAL
==================================================

Crear método agregador:

get_operational_automation_console($business_id, $user_id = null)

- flags (41A)
- escalation state (41B)
- recommendations (41C)
- assignments (41D)
