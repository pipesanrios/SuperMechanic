PHASE: 53C
NAME: Mobile-first Optimization

==================================================
OBJETIVO
==================================================

Optimizar la interfaz del sistema para uso en dispositivos móviles,
mejorando legibilidad, interacción y layout responsive.

==================================================
SCOPE
==================================================
- responsive dashboard
- responsive portal cliente
- ajustes CSS mobile-first
- mejoras UX táctiles básicas

==================================================
OUT OF SCOPE
==================================================
- app móvil nativa
- PWA
- rediseño completo UI
- animaciones complejas
- offline mode

==================================================
ALLOWED FILES
==================================================
- assets/css/admin.css
- assets/css/portal.css
- assets/js/portal.js
- assets/js/admin.js si existe
- includes/dashboard/*
- includes/portal/*
- docs/contracts/53C.md
- docs/contracts/validation/53C-validation.md

==================================================
MEJORAS REQUERIDAS
==================================================

1. Layout responsive:
- cards en una columna en móvil
- grids → stack vertical

2. Tipografía:
- tamaños legibles
- evitar overflow

3. Interacción:
- botones táctiles (tap-friendly)
- spacing adecuado

4. Tablas:
- scroll horizontal o conversión a bloques

5. Portal:
- procesos legibles
- documentos accesibles
- historial claro en móvil

==================================================
CRITERIO DE IMPLEMENTACIÓN
==================================================
- mobile-first en CSS donde sea posible
- media queries claras
- no romper desktop
- no duplicar vistas
- mantener performance

==================================================
ACCEPTANCE CRITERIA
==================================================
- usable en móvil real
- sin overflow grave
- navegación clara
- sin regresión desktop

==================================================
VALIDATIONS REQUIRED
==================================================
- php lint
- QA runner
- runtime manual (móvil real o devtools)
