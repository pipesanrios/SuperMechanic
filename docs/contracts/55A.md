PHASE: 55A
NAME: Elementor Widgets (Integración Nativa)

==================================================
OBJETIVO
==================================================
Integrar Elementor como capa visual nativa del sistema creando widgets
que reutilicen shortcodes/controladores existentes sin duplicar lógica.

==================================================
SCOPE
==================================================
- integración Elementor en `includes/integrations/elementor/`
- loader de widgets con hook `elementor/widgets/register`
- registro condicional en plugin (`did_action('elementor/loaded')`)
- widgets nativos mínimos para dashboards y vistas cliente

==================================================
ALLOWED FILES
==================================================
- includes/integrations/elementor/class-elementor-loader.php
- includes/integrations/elementor/widgets/class-client-dashboard-widget.php
- includes/integrations/elementor/widgets/class-client-vehicles-widget.php
- includes/integrations/elementor/widgets/class-client-processes-widget.php
- includes/integrations/elementor/widgets/class-client-quotes-widget.php
- includes/integrations/elementor/widgets/class-client-invoices-widget.php
- includes/integrations/elementor/widgets/class-mechanic-dashboard-widget.php
- includes/class-plugin.php
- docs/contracts/55A.md
- docs/contracts/validation/55A-validation.md

==================================================
OUT OF SCOPE
==================================================
- cambios CRM Pipeline
- lógica de negocio nueva para dashboards/portal
- acceso directo a base de datos desde widgets
- reemplazo de shortcodes actuales

==================================================
ACCEPTANCE CRITERIA
==================================================
- widgets visibles en Elementor cuando está activo
- widgets renderizan contenido real mediante shortcode/controller existente
- no regresión de shortcodes existentes
- sin bypass de permisos/ownership

==================================================
VALIDATIONS REQUIRED
==================================================
- php scripts/php-lint.php --all
- php scripts/qa-runner.php --contract=docs/contracts/validation/55A-validation.md --output=text
- runtime/manual:
  - Elementor activo
  - widgets visibles/arrastrables
  - render de contenido real
  - sin errores JS/PHP
