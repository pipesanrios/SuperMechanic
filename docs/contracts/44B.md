PHASE: 44B
NAME: Edición básica de reglas

==================================================
OBJETIVO
==================================================

Permitir editar configuración básica de reglas operativas
sin tocar DB manualmente.

==================================================
SCOPE
==================================================

- formulario básico por regla
- enabled
- execution_mode
- max_items_auto
- guardado vía repository

==================================================
OUT OF SCOPE
==================================================

- builder avanzado
- edición de config compleja
- cron
- nuevas reglas
- DSL

==================================================
ALLOWED FILES
==================================================

- includes/automation/class-operational-rules-repository.php
- includes/automation/class-operational-rules-service.php
- includes/dashboard/class-admin-dashboard-controller.php
- assets/css/admin.css
- docs/contracts/44B.md
- docs/contracts/validation/44B-validation.md

==================================================
ACCEPTANCE CRITERIA
==================================================

- se puede editar regla
- guardado persistente por business_id
- sin regresión 43E
