PHASE: 55D
NAME: External Connectors Base

==================================================
OBJETIVO
==================================================

Implementar una capa base de conectores externos para
enviar datos del sistema a servicios de automatización,
hojas de cálculo y endpoints comerciales simples.

==================================================
SCOPE
==================================================
- conectores salientes simples
- conectores reutilizables
- dispatch controlado
- payloads consistentes
- integración con eventos ya formalizados

==================================================
OUT OF SCOPE
==================================================
- OAuth complejo
- SDKs oficiales pesados
- conectores bidireccionales complejos
- sync incremental avanzada
- panel visual complejo de mappings
- integraciones enterprise profundas

==================================================
CONECTORES INICIALES
==================================================

Implementar al menos soporte base para:

1. Generic Webhook Connector
- endpoint configurable
- POST JSON

2. Google Sheets Style Connector
- endpoint webhook-style (Apps Script / Make / Zapier)
- payload tabular/simple

3. External Email Trigger Connector
- trigger saliente a endpoint externo o webhook email workflow

==================================================
MODELO BASE
==================================================

Un connector debe tener al menos:
- id
- name
- connector_type
- endpoint_url
- status (active/inactive)
- event_name
- config_json
- created_at
- updated_at

Persistencia:
- tabla propia o WP option estructurada
- preferible tabla si queda limpio

==================================================
SERVICE REQUERIDO
==================================================

Connector_Service

Métodos mínimos:
- get_supported_connector_types()
- create_connector($data)
- update_connector($id, $data)
- delete_connector($id)
- dispatch_to_connector($connector_id, $payload)
- dispatch_event_to_connectors($event_name, $payload)

==================================================
TIPOS SOPORTADOS
