# 56P11-C1 - Shared PDF Engine Loader

## Scope
- Centralize embedded PDF engine loading for shared PDF generation.
- Ensure invoice, quote and reporting PDF exports detect bundled TCPDF in fresh requests.
- Keep PDF generation behavior compatible.

## Allowed Files
- `includes/helpers/class-pdf-service.php`
- `includes/reporting/class-report-pdf-service.php`
- `docs/CURRENT_STATE.md`
- `docs/QA_REPORT.md`

- `docs/contracts/56P11-C1.md`
- `docs/contracts/validation/56P11-C1-validation.md`

## Forbidden
- No calculation changes.
- No layout changes.
- No filename changes.
- No PDF engine replacement.
- No invoice/quote business logic changes.
- No schema changes.

## Acceptance Criteria
- `PDF_Service::can_generate_pdf()` returns true in a fresh request when bundled TCPDF exists.
- Invoice PDF export passes 3/3 in fresh request.
- Quote PDF export passes 3/3 in fresh request.
- Reporting PDF export remains PASS 3/3.
- Invalid invoice ID fails cleanly without `sm_pdf_engine_unavailable`.
- Invalid quote ID fails cleanly without `sm_pdf_engine_unavailable`.
