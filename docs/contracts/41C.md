==================================================
OBJETIVO FUNCIONAL
==================================================

Crear sugerencias automáticas como:

1. si hay tareas overdue:
   - sugerir resolver backlog vencido

2. si hay procesos retrasados:
   - sugerir revisar bloqueos operativos

3. si hay saturación por usuario:
   - sugerir redistribuir carga

4. si hay múltiples señales críticas:
   - sugerir intervención operativa inmediata

5. si workload está vacío pero hay citas próximas:
   - sugerir preparar trabajo programado
