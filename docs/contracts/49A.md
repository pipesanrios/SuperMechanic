PHASE: 49A
NAME: Business Membership Model

==================================================
OBJETIVO
==================================================

Crear el modelo base de membresía por negocio para asociar
usuarios a business_id con un rol operativo y estado,
como fundamento para acceso por negocio y transferencias futuras.

==================================================
SCOPE
==================================================

- tabla de membresías por negocio
- installer + repository + service
- lectura centralizada de membresías
- soporte para rol operativo por negocio
- soporte para negocio principal

==================================================
OUT OF SCOPE
==================================================

- UI de administración
- transferencias
- super admin global
- auditoría avanzada
- cambios de lógica de negocio
- CRM Pipeline
- migraciones masivas complejas
