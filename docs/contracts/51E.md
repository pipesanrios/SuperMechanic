PHASE: 51E
NAME: Data Export Base

==================================================
OBJETIVO
==================================================

Implementar una base de exportación de datos para facilitar
portabilidad, respaldo operativo y preparación comercial del producto.

==================================================
SCOPE
==================================================
- exportación admin básica
- formatos iniciales seguros
- selección por tipo de dataset
- exportación desacoplada desde services/repositories

==================================================
OUT OF SCOPE
==================================================
- importación
- exportación masiva incremental
- backups completos del sistema
- compresión/zip avanzada
- exportación programada
- CRM Pipeline complejo nuevo

==================================================
ALLOWED FILES
==================================================
- includes/export/class-export-service.php (nuevo)
- includes/admin/class-export-admin-controller.php (nuevo)
- includes/class-plugin.php
- docs/contracts/51E.md
- docs/contracts/validation/51E-validation.md

==================================================
DATASETS INICIALES
==================================================
Soportar al menos:
- clients
- vehicles
- processes
- memberships
- notifications

==================================================
FORMATOS
==================================================
Inicialmente:
- JSON
- CSV (solo si queda limpio; si no, JSON primero)

==================================================
SERVICE REQUERIDO
==================================================

Export_Service

Métodos mínimos:
- export_dataset($dataset_key, $format = 'json')
- get_supported_datasets()
- build_export_payload($dataset_key)

==================================================
CRITERIO DE IMPLEMENTACIÓN
==================================================
- capability: sm_manage_plugin
- nonce obligatorio
- exportación segura
- no hardcodear en controller
- usar payloads limpios y consistentes

==================================================
UI REQUERIDA
==================================================
Página admin:
- Super Mechanic → Export

Mostrar:
- datasets disponibles
- formato
- botón export por dataset

==================================================
ACCEPTANCE CRITERIA
==================================================
- página Export visible
- exportación JSON funciona
- datasets soportados exportan correctamente
- sin regresión admin
