PHASE: 51C
NAME: Plan Limits / Pricing Base

==================================================
OBJETIVO
==================================================

Implementar una base de límites por plan para que la licencia
tenga efecto funcional sobre el producto, preparando monetización real.

==================================================
SCOPE
==================================================
- catálogo de planes
- límites por plan
- service central de plan limits
- validación de consumo contra límites
- estado visible en admin/license

==================================================
OUT OF SCOPE
==================================================
- billing real
- checkout
- validación remota de suscripción
- upgrade/downgrade automático
- bloqueo agresivo de funcionalidades
- UI comercial pública
