PHASE: 46B
NAME: Strategic caching

==================================================
OBJETIVO
==================================================

Reducir carga de cómputo mediante caching selectivo (transients / runtime cache)
en datos no críticos del dashboard.

==================================================
SCOPE
==================================================

- caching de métricas agregadas
- caching de KPIs
- caching de summaries
- caching por request cuando aplique

==================================================
OUT OF SCOPE
==================================================

- caching de acciones operativas
- caching de workload individual
- caching de ejecución de reglas
- caching de logs en tiempo real
- lazy loading
- indexación DB

==================================================
ALLOWED FILES
==================================================

- includes/dashboard/class-workload-service.php
- includes/dashboard/class-admin-dashboard-controller.php
- includes/automation/class-operational-rules-service.php (solo lectura agregada)
- docs/contracts/46B.md
- docs/contracts/validation/46B-validation.md

==================================================
CRITERIO DE IMPLEMENTACIÓN
==================================================

- usar transients de WordPress
- TTL corto (30s–120s)
- invalidación simple (no compleja)
- fallback seguro si cache no existe
- no romper consistencia funcional

==================================================
ACCEPTANCE CRITERIA
==================================================

- menos recomputación en dashboard
- mejora adicional en tiempo de carga
- datos siguen siendo coherentes
- sin impacto en acciones operativas

==================================================
VALIDATIONS REQUIRED
==================================================

- php lint
- QA runner
- runtime manual

==================================================
TECHNICAL DEBT
==================================================

- invalidación avanzada no implementada
- cache por usuario no implementado
