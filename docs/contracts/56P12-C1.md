  - `body_type`
  - `fuel_type`
  - `transmission`
  - `engine`
- Vehicle reads return the new fields.
- Vehicle admin create/edit can populate the new fields from selected catalog record.
- User can manually override visible technical fields before save.
- Selected catalog ID is accepted only when it belongs to the current business.
- Existing vehicle create/edit remains backward compatible.
