PHASE: 54D
NAME: Commercial Readiness

==================================================
OBJETIVO
==================================================

Preparar el sistema para demostración comercial y presentación
a clientes mediante dataset demo, modo demo y ocultamiento de
datos sensibles en contextos de showcase.

==================================================
SCOPE
==================================================
- demo mode base
- dataset demo / seed canónico
- ocultamiento de datos sensibles
- señales visuales de entorno demo
- preparación para presentaciones

==================================================
OUT OF SCOPE
==================================================
- multi-demo manager
- generador masivo de datasets
- permisos comerciales complejos
- sandbox multitenant aislado
- tour guiado interactivo

==================================================
ALLOWED FILES
==================================================
- includes/demo/class-demo-service.php (nuevo)
- includes/admin/class-demo-admin-controller.php (nuevo)
- scripts/seed-demo-commercial.php (nuevo o consolidar uno canónico)
- includes/class-plugin.php
- assets/css/admin.css
- docs/contracts/54D.md
- docs/contracts/validation/54D-validation.md

==================================================
FUNCIONALIDAD REQUERIDA
==================================================

1. Demo mode:
- enable / disable
- persistencia simple en WP options

2. Demo dataset:
- negocio demo
- clientes demo
- vehículos demo
- procesos demo
- quotes/invoices/payments demo si aplica

3. Protección visual:
- badge/banner "Demo Mode"
- ocultamiento o enmascarado de datos sensibles si aplica

4. Utilidad comercial:
- entorno limpio y presentable
- no depender de datos reales

==================================================
SERVICE REQUERIDO
==================================================

Demo_Service

Métodos mínimos:
- is_demo_mode()
- enable_demo_mode()
- disable_demo_mode()
- get_demo_state()
- mask_sensitive_value($value)

==================================================
UI REQUERIDA
==================================================

Página admin:

- resumen del dataset demo
