PHASE: 54A
NAME: Reporting Core

==================================================
OBJETIVO
==================================================

Implementar una capa base de reporting comercial y operativo
con métricas reales del sistema, desacopladas y reutilizables.

==================================================
SCOPE
==================================================
- reporting service
- reporting repository
- métricas agregadas reales
- cálculo por business_id
- período simple de consulta

==================================================
OUT OF SCOPE
==================================================
- charts complejos
- filtros avanzados UI
- exportación avanzada
- PDFs
- BI histórico profundo
- forecasting

ALLOWED FILES
==================================================

FUENTE DE VERDAD
==================================================

MÉTRICAS REQUERIDAS
==================================================

==================================================
SERVICE REQUERIDO
==================================================

Reporting_Service

Métodos mínimos:
- get_reporting_summary($business_id = 0, $range = '30d')

REPOSITORY REQUERIDO
==================================================

Reporting_Repository

- consultas agregadas
- COUNT / SUM / AVG
- filtros por business_id
- filtro simple por rango de fechas si aplica
