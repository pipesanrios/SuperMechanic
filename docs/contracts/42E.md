PHASE: 42E
NAME: Reglas Ejecutables Configurables

==================================================
OBJETIVO
==================================================

Introducir una capa de reglas operativas configurables para preparar
automatizacion futura sin ejecutar acciones automaticamente.

==================================================
SCOPE
==================================================

- definir modelo de reglas operativas (in-memory)
- evaluar reglas usando capas existentes de consola, bulk y asignaciones
- exponer overview de reglas desde Workload_Service
- renderizar bloque dashboard "Reglas Operativas" en modo solo lectura

==================================================
OUT OF SCOPE
==================================================

- cron
- auto ejecucion
- triggers automaticos
- persistencia compleja de reglas
- UI avanzada de edicion
- cambios en CRM Pipeline

  - get_operational_rules($business_id)
