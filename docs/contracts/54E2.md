PHASE: 54E.2
NAME: Embedded TCPDF Install

==================================================
OBJETIVO
==================================================
Integrar TCPDF fisicamente dentro del plugin para habilitar
la descarga PDF en Reporting sin depender de plugins externos.

==================================================
SCOPE
==================================================
- instalar TCPDF embebido en ruta estable del plugin
- mantener flujo actual de Reporting PDF
- mantener fallback claro si la carga falla

==================================================
ALLOWED FILES
==================================================
- includes/reporting/class-report-pdf-service.php
- includes/admin/class-reporting-admin-controller.php (solo si aplica)
- includes/libs/pdf/tcpdf/*
- vendor/tecnickcom/tcpdf/* (alternativo)
- docs/contracts/54E2.md

==================================================
OUT OF SCOPE
==================================================
- cambios CRM Pipeline
- cambios de schema
- cambios en Reporting_Service
- dependencias de plugins externos WordPress

==================================================
VALIDATIONS REQUIRED
==================================================
- php scripts/php-lint.php --all
- runtime manual de descarga/render PDF

==================================================
ACCEPTANCE CRITERIA
==================================================
- boton de descarga visible
- PDF descargable con TCPDF embebido
- contenido HTML renderizado (no texto plano CSS/HTML)
- sin regresion admin/reporting
