PHASE: 46E
NAME: Internal profiling

==================================================
OBJETIVO
==================================================

Agregar profiling interno básico para medir tiempos de render
y generación de datos en dashboard, automation center y logs.

==================================================
SCOPE
==================================================

- medición de tiempos por bloque
- medición total por request
- profiling visible solo para admin autorizado
- sin dependencia externa

==================================================
OUT OF SCOPE
==================================================
- APM externo
- trazas distribuidas
- profiling SQL detallado por query
- analytics avanzada
- exportación de métricas

==================================================
ALLOWED FILES
==================================================
- includes/dashboard/class-admin-dashboard-controller.php
- includes/dashboard/class-workload-service.php
- includes/automation/class-admin-logs-controller.php si existe
- includes/automation/class-operational-rules-service.php
- assets/css/admin.css
- docs/contracts/46E.md
- docs/contracts/validation/46E-validation.md

==================================================
CRITERIO DE IMPLEMENTACIÓN
==================================================
- profiling simple y mantenible
- activable solo con flag seguro
- no visible para usuarios normales
- no cambiar lógica funcional
- no introducir overhead fuerte

==================================================
ACCEPTANCE CRITERIA
==================================================
- profiling disponible para admin
- tiempos visibles por bloque o sección
- no regresión funcional
- overhead bajo

==================================================
VALIDATIONS REQUIRED
==================================================
- php lint
- QA runner
- runtime manual

==================================================
TECHNICAL DEBT
==================================================
- sin profiling SQL fino
- sin almacenamiento histórico
- sin gráficos
