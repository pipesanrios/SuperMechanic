# GETTING_STARTED.md

## Purpose

This guide helps you start using Super Mechanic quickly.

It is intended for:
- workshop owners
- administrators
- new users

Goal:
Have the system running and usable in the shortest time possible.

---

## Step 1 — Install the Plugin

1. Upload the plugin to:
   `/wp-content/plugins/super-mechanic`
2. Go to WordPress Admin → Plugins
3. Activate **Super Mechanic**

---

## Step 2 — Initial Setup

After activation:

- The system will automatically:
  - create required database tables
  - assign roles and capabilities

👉 No manual setup required at this stage.

---

## Step 3 — Create Your First Client

1. Go to **Clients**
2. Click **Add New**
3. Enter client information
4. Save

---

## Step 4 — Add a Vehicle

1. Go to **Vehicles**
2. Click **Add Vehicle**
3. Link it to a client
4. Save

---

## Step 5 — Create a Process

1. Go to **Processes**
2. Click **New Process**
3. Select type:
   - Maintenance
   - Delivery
   - Paperwork
4. Save

---

## Step 6 — Use the Dashboard

Go to:

→ Dashboard → **"Mi trabajo"**

You will see:

- 🔴 Critical items
- 🟡 Attention items
- ⚪ Normal items

👉 This is your daily control center.

---

## Step 7 — Start Working

From here you can:

- manage tasks
- update processes
- follow alerts
- handle appointments

---

## Recommended First Actions

To get familiar with the system:

1. Create 2–3 clients
2. Add vehicles
3. Create processes
4. Add tasks
5. Explore the dashboard

---

## Key Concept

Everything in the system is connected:

- Clients → Vehicles → Processes → Tasks → Alerts

---

## Best Practice

- Always update the system in real time
- Do not leave tasks open after completion
- Use the dashboard daily

---

## Common Issues

### "I don’t see anything on the dashboard"

Possible reason:
- no tasks or processes created yet

Solution:
- create data first (clients, vehicles, processes)

---

### "Nothing appears urgent"

Possible reason:
- no overdue tasks

Solution:
- create tasks with due dates
