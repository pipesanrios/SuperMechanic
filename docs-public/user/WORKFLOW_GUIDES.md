## Purpose

This document shows real-life workflows using Super Mechanic.

It helps users understand:
- how to use the system step by step
- how different modules work together

---

## Workflow 1 — Vehicle Maintenance

### Scenario
A client brings a vehicle for maintenance.

---

### Step 1 — Register Client (if new)

- Go to Clients
- Create new client

---

### Step 2 — Register Vehicle

- Go to Vehicles
- Add vehicle
- Link to client

---

### Step 3 — Create Process

- Create a new process
- Select type: Maintenance

---

### Step 4 — Diagnosis

- Add notes
- Identify issues

Example:
- oil change needed
- brake pads worn

---

### Step 5 — Additional Findings

- Add new notes if more issues are found

Example:
- battery weak
- filter dirty

---

### Step 6 — Approval (if needed)

- Wait for client confirmation
- Update process status

---

### Step 7 — Perform Work

- Complete maintenance tasks
- Update process steps

---

### Step 8 — Final Review

- Check all work completed
- Confirm no pending issues

---

### Step 9 — Mark as Completed

- Set process as finished
- Vehicle ready for pickup

---
