# DASHBOARD.md

## Purpose

Explain how to use the dashboard effectively.

---

## What Is The Dashboard

The dashboard is your main workspace.

It shows everything you need to work daily.

---

## Where To Find It

Go to:

→ Dashboard → "Mi trabajo"

---

## What You Will See

The dashboard includes:

- tasks
- alerts
- processes
- appointments

---

## Priority System

Items are grouped by priority:

### 🔴 Critical

- overdue tasks
- delayed processes
- urgent alerts

👉 Must be handled immediately

---

### 🟡 Attention

- upcoming deadlines
- pending approvals
- follow-ups

👉 Handle during the day

---

### ⚪ Normal

- routine tasks
- non-urgent items

👉 Handle when time allows

---

## How To Use It

1. open dashboard
2. review Critical items
3. move to Attention
4. finish with Normal

---

## Daily Workflow

Start every day here.

Do not jump to modules first.

---

## Key Rule

The dashboard tells you:

👉 what to do next

---

## Best Practices

- check dashboard first thing every day
- do not ignore critical items
- keep tasks updated
- resolve alerts quickly

---

## Common Mistakes

- skipping dashboard
- ignoring alerts
- working outside the system

---

## Final Note

If used correctly, the dashboard becomes:

👉 your operational command center
