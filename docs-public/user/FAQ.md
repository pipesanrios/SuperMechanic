# FAQ.md

## Purpose

Answer common questions and issues when using Super Mechanic.

---

## General Questions

### What is Super Mechanic?

A system to manage workshop operations, vehicles, clients, processes, and tasks.

---

### Do I need technical knowledge?

No. The system is designed for daily operational use.

---

## Usage Questions

### Where do I start each day?

Go to:

→ Dashboard → "Mi trabajo"

---

### Why don’t I see anything on the dashboard?

You need to create:

- clients
- vehicles
- processes
- tasks

---

### What are alerts?

Automatic signals generated by the system.

Examples:
- overdue tasks
- delayed processes

---

### Do I create alerts manually?

No. Alerts are automatic.

---

### What is a process?

A workflow for a vehicle.

Examples:
- maintenance
- delivery
- paperwork

---

### What is the difference between tasks and processes?

- Process → full workflow
- Task → specific action

---

## Technical Questions

### Why is something marked as overdue?

Because the due date has passed.

---

### Why can’t I access some data?

Possible reasons:

- permissions
- ownership restrictions

---

## Troubleshooting

### Changes are not saved

- check internet connection
- refresh page
- try again

---

### I can’t find a client or vehicle

- use search
- check spelling
- verify filters

---

### The system feels empty

You need to add initial data.

---

## Best Practices

- update data in real time
- close tasks when done
- follow process steps
- use dashboard daily

---

## Support Tip

If something doesn’t behave as expected:

1. check tasks
2. check processes
3. check alerts

The system usually indicates what needs attention.

---

## Final Note

Super Mechanic works best when:

👉 everything is recorded in the system

If it’s not in the system, it doesn’t exist.

---

## When Should I Contact Support?

You should contact support if:

- the system shows errors
- data is missing unexpectedly
- actions do not save correctly

---

## What To Check Before Contacting Support

1. refresh the page  
2. verify filters  
3. check tasks and processes  
4. confirm data exists  

---

If the issue persists, then contact support.
